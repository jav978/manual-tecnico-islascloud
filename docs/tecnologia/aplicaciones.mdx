## - Manual técnico de instalación de VSCode

### Introducción

**VSCode** es un editor de código moderno y potente que ofrece una amplia gama de funciones y extensiones. Este manual técnico te mostrará cómo instalar VSCode en tu sistema.

### Requisitos

Para instalar VSCode, necesitarás:

-   Un sistema operativo compatible: Windows, macOS o Linux.
-   Una cuenta de Microsoft (opcional)

### Instalación

Para instalar VSCode, sigue estos pasos:

1. Ve a la página web de VSCode: https://code.visualstudio.com/
2. Haz clic en el botón "Download".
3. Selecciona la versión de VSCode que quieras instalar.
4. Sigue las instrucciones de instalación.

Una vez que hayas instalado VSCode, podrás abrirlo haciendo clic en el icono del programa.

### Configuración

VSCode viene configurado de forma predeterminada con algunas extensiones útiles. Sin embargo, puedes personalizar la configuración de VSCode a tu gusto. Para ello, sigue estos pasos:

1. Abre VSCode.
2. Haz clic en el botón "File" y luego en "Preferences".
3. En la ventana de preferencias, selecciona la pestaña "Extensions".
4. Desde aquí, puedes instalar nuevas extensiones, desinstalar extensiones existentes o modificar la configuración de las extensiones que ya tienes instaladas.

### Recursos adicionales

-   Documentación oficial de VSCode: https://code.visualstudio.com/docs/
-   Comunidad de VSCode: https://code.visualstudio.com/community/

## - Manual técnico de instalación de Docker

### Introducción

Docker es un contenedorizador de aplicaciones que permite a los desarrolladores crear y ejecutar aplicaciones de forma portable y escalable. Este manual técnico te mostrará cómo instalar Docker en tu sistema.

### Requisitos

Para instalar Docker, necesitarás:

-   Un sistema operativo compatible: Windows, macOS o Linux.

### Instalación

Para instalar Docker, sigue estos pasos:

1. Ve a la página web de Docker: https://www.docker.com/
2. Haz clic en el botón "Get Started".
3. Selecciona el tipo de instalación que quieras realizar.
4. Sigue las instrucciones de instalación.

Una vez que hayas instalado Docker, podrás iniciarlo haciendo clic en el icono del programa.

### Configuración

Antes de poder utilizar Docker, deberás configurarlo. Para ello, sigue estos pasos:

1. Inicia Docker.
2. Abre una terminal.
3. Escribe el siguiente comando:

> **`docker run hello-world`**

Este comando ejecutará la imagen de ejemplo de Docker. Si todo funciona correctamente, verás el siguiente mensaje:

> **Hello from Docker!**

**Creación de una imagen de Docker**

Para crear una imagen de Docker, deberás escribir un archivo Dockerfile. Un Dockerfile es un archivo de texto que contiene instrucciones sobre cómo crear una imagen.

Un ejemplo de Dockerfile es el siguiente:

```js 
FROM ubuntu:latest

RUN apt-get update && apt-get install -y nginx

EXPOSE 80

CMD ["nginx", "-g", "daemon off;"]
```

Este Dockerfile crea una imagen de Docker que instala el servidor web Nginx.

**Para crear una imagen de Docker a partir de un Dockerfile, sigue estos pasos:**

1. Escribe un archivo Dockerfile.
2. Guarda el archivo Dockerfile en una carpeta.
3. Abre una terminal.
4. Dirígete a la carpeta que contiene el archivo Dockerfile.
5. Escribe el siguiente comando:

```js
docker build -t <nombre-de-imagen> .

Este comando creará una imagen de Docker con el nombre que especifiques.

### Ejecución de una imagen de Docker

Para ejecutar una imagen de Docker, sigue estos pasos:

1. Inicia Docker.
2. Abre una terminal.
3. Escribe el siguiente comando:

docker run <nombre-de-imagen>
```
Este comando ejecutará la imagen de Docker que especifiques.

### Recursos adicionales

* Documentación oficial de Docker: https://docs.docker.com/
* Comunidad de Docker: https://community.docker.com/

## -Manual técnico de instalación de Node.js ( Windows,MacOs, Linux )

### Introducción

Node.js es un entorno de ejecución multiplataforma, de código abierto, para la capa del servidor basado en el lenguaje de programación JavaScript. 

### Requisitos

* Sistema operativo: Windows, macOS o Linux.
* Memoria: Al menos 1 GB de memoria RAM.
* Espacio en disco: Al menos 50 MB de espacio libre en disco.

Resaltamos algunos detalles adicionales de cada sistema operativo :

* **Windows**: Node.js requiere Windows 7 o posterior. Se recomienda utilizar la última versión de Windows.
* **macOS**: Node.js requiere macOS 10.10 o posterior. Se recomienda utilizar la última versión de macOS.
* **Linux**: Node.js es compatible con la mayoría de las distribuciones de Linux. Se recomienda utilizar la última versión de la distribución.

---

### Instalación

**1. Descargue el instalador :**

El primer paso es descargar el instalador de Node.js desde el sitio web oficial. Puede encontrar el instalador para su sistema operativo en la siguiente página:

> https://nodejs.org/en/download/

![node](/img/aplicaciones/nodejs.png "Node")

**2. Ejecute el instalador**

Una vez que haya descargado el instalador, ejecútelo para iniciar el proceso de instalación. El instalador le guiará a través de los pasos necesarios para instalar Node.js.

**3. Verifique la instalación**

Una vez que el instalador haya finalizado, puede verificar la instalación ejecutando el siguiente comando en la línea de comandos:

> **`node -v`**

**4- Consideraciones de instalacion segun el sistema operativo :**

**Windows**

En Windows, puede instalar Node.js utilizando el instalador MSI. Para hacerlo, siga estos pasos:

* Descargue el instalador MSI desde el sitio web oficial.
* Haga doble clic en el archivo de instalación MSI para iniciar el proceso de instalación.
* Siga las instrucciones del instalador para completar la instalación.

**MacOs**

En macOS, puede instalar Node.js utilizando el instalador DMG. Para hacerlo, siga estos pasos:

* Descargue el instalador DMG desde el sitio web oficial.
* Haga doble clic en el archivo de instalación DMG para abrirlo.
* Arrastre el icono de Node.js a la carpeta "Aplicaciones".

**Linux**

En Linux, puede instalar Node.js utilizando el administrador de paquetes de su distribución. Por ejemplo, en Ubuntu, puede instalar Node.js ejecutando el siguiente comando:

> **`sudo apt install nodejs`**

Para verificar la instalación, puede ejecutar el siguiente comando:

> **`nvm ls`**

La salida del comando mostrará las versiones de Node.js que están instaladas en su sistema.

## - Manual técnico de instalación de Angular CLI 

### Introduccion 

**Angular CLI** es una herramienta de línea de comandos que facilita el desarrollo de aplicaciones Angular. Proporciona una serie de comandos que se pueden utilizar para crear, compilar, probar y desplegar aplicaciones Angular de forma rápida y sencilla.

### Requisitos

Los requisitos para instalar Angular CLI son los siguientes:

* Sistema operativo: Windows, macOS o Linux.
* Memoria: Al menos 1 GB de memoria RAM.
* Espacio en disco: Al menos 50 MB de espacio libre en disco.

### Instalación 


Para instalar **Angular CLI**, primero debe instalar **Node.js**. Una vez que Node.js esté instalado, puede instalar Angular CLI ejecutando el siguiente comando en la línea de comandos:

> **`npm install -g @angular/cli`**

![angular](/img/aplicaciones/angularcli.png "angular")

Este comando instalará Angular CLI globalmente, lo que significa que estará disponible en cualquier directorio de su sistema.

Para verificar la instalación, puede ejecutar el siguiente comando:

> **`ng --version`**

![angular](/img/aplicaciones/angular.png "angular")


Si el comando se ejecuta correctamente, Angular CLI está instalado y la versión se mostrará en la salida del comando.

**Pasos específicos para cada sistema operativo**

Los pasos específicos para instalar Angular CLI pueden variar según el sistema operativo que esté utilizando.

**Windows**

En Windows, puede instalar Angular CLI utilizando el administrador de paquetes de Node.js. Para hacerlo, siga estos pasos:

* Abra una ventana de línea de comandos.

* Ejecute el siguiente comando para instalar el administrador de paquetes de Node.js:

> **`npm install -g npm`**

* Ejecute el siguiente comando para instalar Angular CLI:

> **`npm install -g @angular/cli`**

**MacOS**

En macOS, puede instalar Angular CLI utilizando el administrador de paquetes de Node.js. Para hacerlo, siga estos pasos:

* Abra una ventana de terminal.

* Ejecute el siguiente comando para instalar el administrador de paquetes de Node.js:

> **`npm install -g npm`**

* Ejecute el siguiente comando para instalar Angular CLI:

> **`npm install -g @angular/cli`**

**Linux**

En Linux, puede instalar Angular CLI utilizando el administrador de paquetes de Node.js. Para hacerlo, siga estos pasos:

* Abra una terminal.

* Ejecute el siguiente comando para instalar el administrador de paquetes de Node.js:

> **`sudo apt install npm`**

* Ejecute el siguiente comando para instalar Angular CLI:

> **`npm install -g @angular/cli`**
