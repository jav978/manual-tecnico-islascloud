### Ingresar a MariaDB con phpmyAdmin.

:::tip Ingresar con docker

Es posible acceder a MariaDB a través de un contenedor Docker utilizando la dirección IP del servidor donde está alojada la base de datos.

## Levantar contenedor

Al levantar el contenedor Docker con la imagen de MariaDB, exponemos el puerto 3306 para permitir la conexión remota.

## Conectarse al contenedor

Con la dirección IP del servidor y las credenciales correctas, podemos conectarnos desde cualquier cliente SQL como MySQL Workbench. 

## Gestionar la base de datos

Una vez conectados dentro del contenedor Docker, tenemos acceso completo a la base de datos MariaDB. Esto nos permite:

- Ejecutar consultas SQL
- Crear y modificar tablas y registros
- Importar y exportar datos
- Respaldar y restaurar bases de datos 
- Gestionar usuarios y permisos

Entre otras funciones de administración y gestión de la base de datos ahora alojada en un contenedor Docker portable y aislado.
:::
