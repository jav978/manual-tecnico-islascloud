# Portainer 

<Highlight color="#666">Tecnologías  </Highlight>

## Introducción

Portainer es una plataforma de gestión de contenedores Docker con interfaz gráfica web. Permite administrar contenedores Docker de forma visual e intuitiva.

## Funcionamiento de Portainer

- Portainer se instala como un contenedor Docker adicional dentro del entorno Docker existente. No reemplaza ni modifica la instalación de Docker.

- Una vez instalado y ejecutándose, Portainer expone un servidor web en el puerto 9000. Se accede vía navegador web para usar la interfaz.

- Portainer se conecta al demonio Docker local y obtiene toda la información sobre imágenes, contenedores, volúmenes, redes, etc. Lo presenta de forma gráfica. 

- Permite gestiones como iniciar/detener contenedores, ver logs en tiempo real, monitorizar el uso de recursos, acceder dentro de los contenedores, entre otros.

- También facilita la administración de imágenes, volúmenes, redes, usuarios y roles.

- Admite la integración con Docker Swarm para administrar clústeres Docker. 

- La interfaz es responsive y accesible desde cualquier dispositivo. No requiere instalación de software adicional.

## Procedimiento para habilitar o deshabilitar los contenedores 


>Dentro del módulo podemos ver las siguiente características: 

tiene un conjunto de botones con diversas acciones para gestionar el comportamiento de los contenedores de las cuales son:

* (**Start**) inicia el contenedor
* (**Stop**)  detiene el contenedor
* (**Kill**) eliminar el contenedor
* (**Restart**) reinicia el contenedor
* (**Pause**) pausa el contenedor
* (**Resume**) muestra un resumen del contenedor
* (**Remove**) borra el contenedor 
* (**Add Container**) Agrega un nuevo contenedor.  

![Influxd](/img/portainer/portainer-6.png "portainer")  


:::info Gestion de contenedores Portainer

**Start**: Se inicia el contenedor o conjunto de contenedores. muestra un mensaje de color verde con el estado running.

**Stop**: Botón para detener el contenedor. muestra una mensaje de color rojo con el mensaje stopped.

**Pause**: Botón para pausar el contenedor. como se en la imagen muestra una leyenda de color amarillo con el mensaje paused.

**Resumen**: Botón para despausar el contenedor. como se en la imagen muestra una leyenda de color verde con el mensaje running.
:::
